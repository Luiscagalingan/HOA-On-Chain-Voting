# HOA-On-Chain-Voting

One-Line Description
Token-weighted on-chain voting for HOA elections and policy proposals.
Track
Track 2 — Financial Inclusion & Everyday Payments
Problem It Solves
HOA elections in subdivisions are frequently contested and lack transparency. Residents have no way to independently verify vote counts, and results can be disputed or manipulated without a trustless audit trail.
Solution
Token-weighted on-chain voting for HOA elections and policy proposals, ensuring every vote is verifiable and tamper-proof.
How It Uses Stellar
•	Classic Asset — Used as the voting token distributed to eligible HOA members.
•	Soroban Smart Contract — Handles vote tallying, proposal state management, and result finalization on-chain.
